### LoadConfig Implementation
- Created Go module structure (github.com/zkwentz/amazon-cli)
- Created internal/config directory
- Implemented Config struct with three sub-structs:
  - AuthConfig: Holds access_token, refresh_token, and expires_at
  - DefaultsConfig: Holds address_id, payment_id, and output_format
  - RateLimitConfig: Holds min_delay_ms, max_delay_ms, and max_retries
- Implemented LoadConfig(path string) (*Config, error) function:
  - Uses default path (~/.amazon-cli/config.json) if path is empty
  - Creates directory with 0700 permissions if it doesn't exist
  - Returns default config if file doesn't exist (first run scenario)
  - Parses JSON and returns populated Config struct for existing files
  - Proper error handling for all failure cases
- Implemented getDefaultConfig() helper function with sensible defaults:

  - Min delay: 1000ms
  - Max delay: 5000ms
  - Max retries: 3
