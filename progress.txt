## Task Completed: Create pkg/models/subscription.go

### What Was Implemented:

1. **Created pkg/models/subscription.go** with the following data models:
   - `Subscription` struct: Represents a Subscribe & Save subscription with fields:
     - SubscriptionID: Unique identifier for the subscription
     - ASIN: Amazon product identifier
     - Title: Product name
     - Price: Subscription price
     - DiscountPercent: Subscribe & Save discount percentage
     - FrequencyWeeks: Delivery frequency in weeks
     - NextDelivery: Date of next scheduled delivery
     - Status: Current subscription status (active, paused, cancelled)
     - Quantity: Number of items per delivery

   - `SubscriptionsResponse` struct: Response wrapper containing array of subscriptions

   - `UpcomingDelivery` struct: Represents an upcoming subscription delivery with:
     - SubscriptionID: Reference to parent subscription
     - ASIN: Product identifier
     - Title: Product name
     - DeliveryDate: Scheduled delivery date
     - Quantity: Number of items in delivery

2. **Created comprehensive tests in pkg/models/subscription_test.go**:
   - TestSubscriptionJSONMarshaling: Verifies JSON serialization/deserialization of Subscription struct

   - TestSubscriptionJSONTags: Verifies correct JSON field naming (snake_case)
   - TestUpcomingDeliveryJSONTags: Validates JSON tags for upcoming deliveries
   - TestSubscriptionWithDifferentStatuses: Tests various subscription statuses
