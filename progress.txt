### Command Output Examples
No authentication:
```json
{
  "authenticated": false
}
```

Valid authentication:
```json
{
  "authenticated": true,
  "expired": false,
  "expires_at": "2026-01-19T12:00:00Z",
  "expires_in_seconds": 57645
}
```

Expired authentication:
```json
{
  "authenticated": true,
  "expired": true,
  "expires_at": "2020-01-19T12:00:00Z",
  "expires_in_seconds": -189331161
}
```
